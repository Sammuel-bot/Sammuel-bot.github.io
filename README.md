# Samuel Agyapong — Personal Portfolio

Live site: [sammuel-bot.github.io](https://sammuel-bot.github.io)

---

## About

Personal portfolio website for Samuel Agyapong, Electronics Engineering Technology student at Grambling State University. Built to showcase hardware engineering projects, technical skills, and research experience to recruiters and collaborators.

---

## Built With

- **React 18** + **Vite** — component-based UI and build tooling
- **Tailwind CSS** — utility-first styling
- **Framer Motion** — animations and scroll transitions
- **React Icons** — icon library
- **GitHub Pages** — static hosting

---

## Sections

- **Hero** — animated particle canvas, typing animation, glitch name effect
- **About** — bio, animated terminal, academic profile card
- **Projects** — hardware and software project cards with tech stacks
- **Skills** — grouped by Hardware, Software, Renewable Energy, and EDA Tools
- **Contact** — interactive expandable contact cards

---

## Projects Featured

- FPGA Traffic Light Controller — Verilog HDL, Icarus Verilog, GTKWave
- DWIPS Wearable Safety System — Embedded Logic, PCB Design
- Image Processing Object Detection — Python, OpenCV
- Lightweight Drone Frame Design — AutoCAD
- Line-Following Robot — Arduino, Infrared Sensors
